🔄 Movimiento entre Servicios

Este tipo de movimiento requiere que el servicio que recibe la ficha confirme la recepción en el sistema.

  • SOME envía a Urgencia → Urgencia debe confirmar.
  • Urgencia devuelve a SOME → SOME debe confirmar.

Cada servicio es responsable de registrar cuando recibe la ficha.

📦 Movimiento Controlado desde Origen

La ficha puede circular por distintos servicios, pero solo el servicio clinico de origen confirma la recepción final.

  • SOME/Fichero/Archivo envía a Urgencia.
  • Puede pasar por otros servicios.
  • Cuando regresa a SOME/Fichero/Archivo, allí se confirma la recepción.

Otros servicios no necesitan registrar recepción.
